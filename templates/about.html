General

Origin

This project came about due to my wife asking me to water her plants while she was away. The issue that I ran it to was that I would water them every other day, as instructed, but the plants were slowly dying. To this day I do not know if this was because I watered them too much, or too little.

My remedy to this was to automate as much of it as I can. I started by purchasing a Raspberry Pi development board, a moisture sensor for soil, and a small water pump. I used the Raspberry Pi's built in GPIO to get the moisture sensor's reading on moisture level, and have it trigger the pump when it was dry. Once that was working, I branched out from there.

Technical

Hardware

Currently all hardware that this has been used on or tested with is located under the useful items link in the footer.
It has been tested on a Raspberry Pi 3 B+. It should work on all versions of the Raspberry Pi which support GPIO.

Software

This software is open source and has been built in Python 3.5, as that is the default installation on the Raspberry Pi. This website was built utilizing the Flask framework.
